In \features_meteorological_variables .npy object actually and directly used by models are stored
Here there are also .nc file of the splits

"..._image_sequences.npy" contain the image time series as numpy array for each instance of a set
"..._OHE.npy" contain the One Hot Encoding of each time step of the time series for each instance of a set
"..._image_sequences_dates.npy" contain the date of each time step of the image time series for each instance of a set
